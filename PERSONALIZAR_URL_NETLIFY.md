# Cómo Personalizar la URL en Netlify

Tu app Full Express actualmente está en:
```
https://lucent-alpaca-25d7fe.netlify.app/
```

Puedes cambiarla a algo más memorable como:
```
https://full-express-taxis.netlify.app/
```

## Pasos para Cambiar la URL

1. **Inicia sesión en Netlify**
   - Ve a [app.netlify.com](https://app.netlify.com)
   - Inicia sesión con tu cuenta

2. **Selecciona tu sitio**
   - Haz clic en el sitio "lucent-alpaca-25d7fe"

3. **Ve a Site Settings**
   - En el menú superior, haz clic en "Site settings"

4. **Cambia el nombre del sitio**
   - En la sección "Site information", busca "Site name"
   - Haz clic en "Change site name"
   - Ingresa el nuevo nombre (ej: `full-express-taxis`)
   - Haz clic en "Save"

5. **¡Listo!**
   - Tu nueva URL será: `https://full-express-taxis.netlify.app/`
   - La URL antigua seguirá funcionando y redirigirá automáticamente

## Dominio Personalizado (Opcional)

Si tienes un dominio propio (ej: `fullexpress.com`), puedes configurarlo:

1. En "Site settings", ve a "Domain management"
2. Haz clic en "Add custom domain"
3. Ingresa tu dominio
4. Sigue las instrucciones para configurar los DNS

**Nota:** Netlify proporciona HTTPS gratuito para dominios personalizados.
